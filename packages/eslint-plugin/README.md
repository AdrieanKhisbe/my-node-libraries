# @a-k/eslint-plugin

> Personal configuration for eslint

- Install it with `npm install --save-dev @-k/eslint-plugin` :inbox_tray:

- Set up minimal configuration: :wrench:
    ```json
    {
        "extends": "plugin:@-k/base",
        "plugins": ["@-k"]
    }
    ```
- You're good to go! :wink:
